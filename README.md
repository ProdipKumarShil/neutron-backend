# Contact management Server
* Clone this repo
```
git clone https://github.com/ProdipKumarShil/neutron-backend
```
* Enter ==neutron-backend==
```
cd neutron-backend
```
* Install node packages
```
npm install
```
* Run this server
```
nodemon index.js
```
* Server will hosted in. Go to browser and hit this url
```
http://localhost:5000/
```
